# CHISET (CHISE/Turtle)

CHISE 漢字屬性資料庫的 Turtle RDF 格式匯出。

本倉庫為 [CHISE 專案](https://www.chise.org/) 的 [chiset](https://gitlab.chise.org/CHISE/chiset) GitLab 倉庫之 mirror。

## 資料內容

以 Turtle (`.ttl`) 格式儲存的漢字屬性資料，按康熙部首分檔（R001–R214），包含：

- 跨國字元編碼對照（JIS, GB, KS, CNS, Big5, 大漢和辭典 等）
- 日文讀音（訓讀、音讀、名乘讀）
- 部首、筆畫數、漢語大字典頁碼
- 異體字關係（簡化字、俗字、古字、原字）
